How to use Moab Early App

Open and start using. No login. No account. Everything runs on this computer or device.

Find a deal

The first tab is Deal Browser. Saved deals show here with the most-recent first. Filter by price, team count, or date range. Click Load on any row to open it.

Edit a deal

Click Inputs to change deal terms. Yellow cells are editable. Type a number, hit Tab. Every other tab updates instantly. Use Cmd-Z (or Ctrl-Z) to undo.

Investor Team

Click Investor Team to set up the cap stack. Each member has a Contribution and an Ownership %. Use Group to lock several members to equal share. Use Auto-deduct on one member so their % auto-fills as 100% − everyone else. The cap-stack sum displays in red when it's not 100%, but variants can be forked at any total — the warning is informational.

Fork a variant

Click + New Variant in the top bar. This snapshots the current deal under its current name, then forks: the deal name auto-bumps (v2, then v3, …) and the app switches to the new variant so further edits don't touch the prior version. The first fork will ask permission to write the xlsx snapshot to the folder this app lives in — click Allow once and every future fork writes silently.

Tip: edits between forks are autosaved continuously. + New Variant is for branching — when you want to compare two scenarios side-by-side without overwriting the first.

Make a new deal

Click + New. Pick a template (last-used is highlighted at top). The form opens pre-filled with the template's defaults — edit, save, done.

Settings & Templates

Click the ⚙ gear. Set how many years NOI projects (default 10). Manage templates (Duplicate, Delete, Set as Default, Save current state as new template).

Add to Mac dock or iPhone home screen

iPhone / iPad: Open in Safari → tap Share → Add to Home Screen.
Mac (macOS Sonoma+): Open in Safari → File → Add to Dock. Or in Chrome, click the install icon in the URL bar.

Tooltips

Desktop: hover any button for a brief description.
Mobile: long-press (~½ second) any button for the same.
